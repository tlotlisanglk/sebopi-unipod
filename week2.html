Week 1: Github Repository Setup

Date: 30-9-2025 - 06-10-2025

I created github repository for FabLab documentation.

  • Skills Learned: Good Documentation
  • Digital Tools and Software Used: Visual Studio, Powershell,Command Prompt, Github Page Engine
  • Topics: Version Control
  • Key Concepts: Digital Documentation
  • Challenges Faced: Slow rendering from Github page thus one might think there were no changes committed
 

Introduction

The session introduces FabLab and explains how an individual can create a GitHub repository. It also discusses design thinking and its importance. Finally, it introduces computer-aided design (CAD), covering both 2D and 3D design.

Reflection

The lesson in version control explains the process of documenting your work you do for the future reference and for other people to easily follow what you have learned. It is a long process but very helpful for the experience of work you have done in your career. This documetation can be used to get a job because employers can easily see the relevant work experience they need from you. It was a very refreshing experience and this builds up to the experience of documentation I have done in the past on github.
